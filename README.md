# 七七大王的个人网页

可爱风格的个人主页，展示七七大王的个人信息、兴趣爱好和技能。

## 特性

- 个人介绍展示
- 兴趣爱好展示（AI编程、轻松熊）
- 游戏爱好展示（洛克王国、王者荣耀）
- 技能展示
- 可爱风格设计，融入轻松熊元素
- 响应式布局

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署

构建产物在 `dist` 目录，可以部署到 GitHub Pages 或其他静态网站托管服务。
